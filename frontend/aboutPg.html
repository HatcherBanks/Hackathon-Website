About Us

This website was made for the Troy University Hackathon 2024.

This website's purpose is to track the games one has played along with any relevant information regarding them. One can select the game they are playing and add it to one of three main lists: Currently Playing, Want to Play, and Completed. The user can also create their own lists such as "Horror Games," "Great Games," etc. The user can see their friend's game progress and game reviews and vice versa. The user can also browse an extensive game catalog, and, by clicking on a specific game, can see all relevant information and reviews for that game.
